## 소개

42서울 재학생과 졸업생이 교류하는 온라인 커뮤니티 제작 프로젝트입니다.

이 저장소에는 **프론트엔드 소스코드**가 저장되어 있고, 백엔드 소스코드는 [이곳](https://github.com/42-world/42world-Backend)에서 볼 수 있습니다
